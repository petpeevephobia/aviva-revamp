# A-Viva v2
A static website with cleaner and more scalable code.

## How to run locally for development
1. In the project directory, run the following command. This allows the contact form to be tested.:
```
C:\xampp\php\php.exe -S localhost:8000 -t "C:\code-projects\aviva-revamp"
```
2. Go to [http://localhost:8000/](http://localhost:8000/)
